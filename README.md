### Operating System Requirements

sms-flooding works on any of the following operating systems:<br>
• **Android** (Using the Termux App) <br>
• **Linux**  <br>


------------------------------------------------------------------------

### How to Install 

command for termux.

* `pkg update`

* `pkg upgrade`

* `pkg  install git`

* `pkg  install python3`

* `git clone https://github.com/Rycoder1/Rybomber.git`

* `cd Rybomber`

* `python3 Main.py`


command for Debian.

* `sudo apt update`

* `sudo apt upgrade`

* `sudo install git`

* `git clone https://github.com/Rycoder1/Rybomber.git`

* `Rybomber`

* `python3 Main.py`
